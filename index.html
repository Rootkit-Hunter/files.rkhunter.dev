FILES.RKHUNTER.DEV

Purpose

This site is here to host runtime files for rkhunter; an open source security, monitoring and analyzing tool for POSIX compliant systems.

This repository stores "dynamic" files that can be updated at runtime via the `--update` flag.

The source of this site can be found in the Rootkit-Hunter/files.rkhunter.dev repo on GitHub.

Come and get involved.
